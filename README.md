# 📊 Análise de Dados de Vendas

Este projeto tem como objetivo analisar dados de vendas para identificar padrões de consumo e gerar insights.

---

## 📁 Sobre os dados
Os dados foram criados para fins de estudo e estão armazenados em um arquivo CSV.  
Para as consultas SQL, os dados foram considerados como uma tabela chamada **vendas**.

---

## 🔍 Análises realizadas

- Total de gastos por cliente  
- Produto mais vendido  
- Total de vendas por dia  
- Faturamento total  

---

## 🛠️ Tecnologias utilizadas

- SQL  
- Python (em evolução)  
- Excel  
- Banco de Dados (MySQL / PostgreSQL)  

---

## 💡 Insights encontrados

- O produto mais vendido foi o Perfume  
- Clientes como Ana tiveram maior volume de compras  
- Alguns dias concentraram mais vendas que outros  
- O faturamento varia de acordo com o tipo de produto
